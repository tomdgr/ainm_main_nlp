## ⚠️ Viktige merknader

1. **Delbetalinger (kunder):** `PUT /invoice/:payment` med `paidAmount` < fakturabeløp registrerer automatisk delbetaling. Fakturaen forblir åpen for restbeløpet.
2. **Delbetalinger (leverandør Nilsen AS):** `partialPayment=true` må settes for *begge* utbetalingene (13 700 + 18 650). Avgjør om det er én faktura eller to basert på funn i fase 2b.
3. **Skattetrekk (positiv):** Uvanlig positiv post — behandlet som refusjon. Verifiser kontoplassering (konto 2600) med regnskapsansvarlig.
4. **Bankformat:** `fileFormat` i importsteget må matche bankens eksportformat. Juster om nødvendig (f.eks. "DNB_CSV", "SPAREBANK_CSV" el.l.).
5. **Kontonummer:** Kontonumrene 1920, 8070 og 2600 er basert på norsk standard kontoplan — verifiser mot selskapets faktiske kontoplan i steg 1a.
