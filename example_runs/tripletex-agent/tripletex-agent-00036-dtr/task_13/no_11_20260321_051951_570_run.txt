### Key Design Decisions

| Decision | Rationale |
|---|---|
| **5 days → Mar 16–20** | 5 days ending before today (Mar 21) |
| **`rate: 800` on per diem** | Prompt specifies a daily rate of 800 NOK — set explicitly |
| **`overnightAccommodation: HOTEL`** | Standard for multi-day trips |
| **No `countryCode`** | Domestic trip; including it causes errors |
| **No `count` on costs** | Field doesn't exist on Cost — causes 422 |
| **Everything nested in one POST** | Avoids 409 RevisionException from separate cost creation |
| **Steps 1–5 have dependencies** | Steps 1, 4, 5 are independent (can be batched); Steps 2→3 are sequential; Step 6 depends on all prior results |
