**3a.** `POST /employee/employment`

```json
{
  "employee": { "id": EMPLOYEE_ID },

  "isMainEmployer": true
}
```

**3b.** `POST /employee/employment/details`

```json
{

}
```

**3c.** `POST /division` *(solo si no existe una división)*

```json
{
  "name": "Hovedvirksomhet",
  "startDate": "2026-03-01"
}
```

**3d.** `PUT /employee/employment/{EMPLOYMENT_ID}`
— Vincular la división al empleo.
```json
{
  "id": EMPLOYMENT_ID,
  "version": VERSION,
  "employee": { "id": EMPLOYEE_ID },

  "taxDeductionCode": "loennFraHovedarbeidsgiver",
  "isMainEmployer": true,
  "division": { "id": DIVISION_ID }
}
```
