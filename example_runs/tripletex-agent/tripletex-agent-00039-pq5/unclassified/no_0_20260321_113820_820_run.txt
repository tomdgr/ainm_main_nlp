### Key Partial Payment Handling
- For **customer invoices**: `PUT /invoice/{id}/:payment` with `paidAmount` set to the bank amount. If it's less than `amountOutstanding`, it registers as a partial payment — the invoice remains open with the residual.
- For **supplier invoices**: `POST /supplierInvoice/{id}/:addPayment` with `partialPayment=true` and the exact bank amount. This explicitly allows partial payments; the remaining balance stays outstanding.
