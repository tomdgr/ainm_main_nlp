## PLAN D'EXÉCUTION

**Résumé de la tâche** : Créer et envoyer une facture de 34 100 NOK HT (TVA 25%) au client Lumière SARL (org. 959714320) pour « Stockage cloud ».

---

**1. GET /customer?organizationNumber=959714320** — Rechercher le client Lumière SARL par son numéro d'organisation.
> Récupère l'`id`, `invoiceEmail` et les données du client.

**2. GET /ledger/account?number=1920&fields=id,number,version,bankAccountNumber** — Vérifier que le compte bancaire 1920 a un numéro de compte bancaire enregistré.
> ⚠️ CRITIQUE : Sans `bankAccountNumber` renseigné, la facturation échouera avec l'erreur *« selskapet har ikke registrert et bankkontonummer »*.

**3. (CONDITIONNEL) PUT /ledger/account/{id}** — Si `bankAccountNumber` est vide sur le compte 1920, le mettre à jour AVANT de facturer.
```
Body: {
  "id": <account_id>,
  "version": <account_version>,
  "number": 1920,
  "name": <existing_name>,
  "bankAccountNumber": "86011117947"
}
```

**4. (CONDITIONNEL) PUT /customer/{id}** — Si le champ `invoiceEmail` du client est vide, le renseigner pour permettre l'envoi par email.
```
Body: {
  "id": <customer_id>,
  "version": <customer_version>,
  "invoiceEmail": "<email approprié du client>"
}
```

**5. POST /order** — Créer la commande avec la ligne de commande « Stockage cloud ».
```
Body: {
  "customer": { "id": <customer_id> },
  "orderDate": "2026-03-21",
  "deliveryDate": "2026-03-21",
  "orderLines": [
    {
      "description": "Stockage cloud",
      "unitPriceExcludingVatCurrency": 34100,
      "count": 1,
      "vatType": { "id": 3 }
    }
  ]
}
```
> - `vatType.id = 3` → TVA standard 25% (le montant est HT, la TVA sera calculée en sus)
> - Pas besoin de `product` : on utilise `description` directement
> - `deliveryDate` est obligatoire → date du jour

**6. PUT /order/{id}/:invoice?invoiceDate=2026-03-21&sendToCustomer=true** — Générer la facture depuis la commande ET l'envoyer au client en une seule opération.
> - `invoiceDate` = 2026-03-21 (aujourd'hui)
> - `sendToCustomer=true` → envoi automatique au client
