Når eg har verdiane, vil eg utføre desse 4 kalla i rekkjefølgje:

```
PLAN:
1. GET /department
   — Finn ID for avdelinga nemnde i brevet

2. POST /employee
   — Opprett den tilsette
   Body: {
     firstName, lastName, dateOfBirth, email,
     userType: "STANDARD",
     department: {id: <frå steg 1>}
   }

3. POST /employee/employment
   — Opprett tilsetjingsforhold med stillingsprosent, årslønn
     og arbeidstidsordning inline
   Body: {
     employee: {id: <frå steg 2>},
     startDate: "<startdato>",
     isMainEmployer: true,
     taxDeductionCode: "loennFraHovedarbeidsgiver",
     employmentDetails: [{
       date: "<startdato>",
       employmentType: "ORDINARY",
       employmentForm: "PERMANENT",   ← eller TEMPORARY
       remunerationType: "MONTHLY_WAGE",
       workingHoursScheme: "NOT_SHIFT",
       percentageOfFullTimeEquivalent: <stillingsprosent>,
       annualSalary: <årslønn>
     }]
   }

4. POST /employee/standardTime
   — Konfigurer standard dagleg arbeidstid
   Body: {
     employee: {id: <frå steg 2>},
     fromDate: "<startdato>",
     hoursPerDay: <t.d. 7.5>
   }
```
